# Social
> [!IMPORTANT]
> This is an undocumented Sandboxable capability. If/when the capability receives [official documentation](https://create.roblox.com/docs/scripting/capabilities), this file will be removed!

## Description
Access to API that relates to social actions, such as invites, among other things.

- [`ExperienceNotificationService`](https://create.roblox.com/docs/reference/engine/classes/ExperienceNotificationService)

- `Players` Members:

  - [`Player.GetFriendsOnline`](https://create.roblox.com/docs/reference/engine/classes/Player#GetFriendsOnline) (also requires **Players**)
  - [`Player.IsFriendsWithAsync`](https://create.roblox.com/docs/reference/engine/classes/Player#IsFriendsWithAsync) (also requires **Players**)
  - [`Player.IsFriendsWith`](https://create.roblox.com/docs/reference/engine/classes/Player#IsFriendsWith) (also requires **Players**)
  - [`Players.GetFriendsAsync`](https://create.roblox.com/docs/reference/engine/classes/Players#GetFriendsAsync) (also requires **Players**)
