# Groups
> [!IMPORTANT]
> This is an undocumented Sandboxable capability. If/when the capability receives [official documentation](https://create.roblox.com/docs/scripting/capabilities), this file will be removed!

## Description
Access to API that gets group information, and can prompt users to join groups.

## Assigned API, as of v710
- [`GroupService`](https://create.roblox.com/docs/reference/engine/classes/GroupService)
- `Players` Members:
  - [`Player.GetRankInGroupAsync`](https://create.roblox.com/docs/reference/engine/classes/Player#GetRankInGroupAsync) (also requires **Players**)
  - [`Player.GetRankInGroup`](https://create.roblox.com/docs/reference/engine/classes/Player#GetRankInGroup) (also requires **Players**)
  - [`Player.GetRoleInGroupAsync`](https://create.roblox.com/docs/reference/engine/classes/Player#GetRoleInGroupAsync) (also requires **Players**)
  - [`Player.GetRoleInGroup`](https://create.roblox.com/docs/reference/engine/classes/Player#GetRoleInGroup) (also requires **Players**)
  - [`Player.IsInGroupAsync`](https://create.roblox.com/docs/reference/engine/classes/Player#IsInGroupAsync) (also requires **Players**)
  - [`Player.IsInGroup`](https://create.roblox.com/docs/reference/engine/classes/Player#IsInGroup) (also requires **Players**)
