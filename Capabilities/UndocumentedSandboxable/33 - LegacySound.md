# LegacySound
> [!IMPORTANT]
> This is an undocumented Sandboxable capability. If/when the capability receives [official documentation](https://create.roblox.com/docs/scripting/capabilities), this file will be removed!

## Description
Access to API that are related to the legacy Audio API.

## Assigned API, as of v710
- `AssetSoundEffect`, `ChannelSelectorSoundEffect`, [`ChorusSoundEffect`](https://create.roblox.com/docs/reference/engine/classes/ChorusSoundEffect), [`CompressorSoundEffect`](https://create.roblox.com/docs/reference/engine/classes/CompressorSoundEffect), `CustomSoundEffect`, [`DistortionSoundEffect`](https://create.roblox.com/docs/reference/engine/classes/DistortionSoundEffect), [`EchoSoundEffect`](https://create.roblox.com/docs/reference/engine/classes/EchoSoundEffect), [`EqualizerSoundEffect`](https://create.roblox.com/docs/reference/engine/classes/EqualizerSoundEffect), [`FlangeSoundEffect`](https://create.roblox.com/docs/reference/engine/classes/FlangeSoundEffect), [`PitchShiftSoundEffect`](https://create.roblox.com/docs/reference/engine/classes/PitchShiftSoundEffect), [`ReverbSoundEffect`](https://create.roblox.com/docs/reference/engine/classes/ReverbSoundEffect), [`SoundEffect`](https://create.roblox.com/docs/reference/engine/classes/SoundEffect), [`TremoloSoundEffect`](https://create.roblox.com/docs/reference/engine/classes/TremoloSoundEffect)
- [`Sound`](https://create.roblox.com/docs/reference/engine/classes/Sound), [`SoundGroup`](https://create.roblox.com/docs/reference/engine/classes/SoundGroup)
