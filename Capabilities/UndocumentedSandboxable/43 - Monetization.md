# Monetization
> [!IMPORTANT]
> This is an undocumented Sandboxable capability. If/when the capability receives [official documentation](https://create.roblox.com/docs/scripting/capabilities), this file will be removed!

## Description
Access to API that relate to live merchandise, or things that directly relate to real life currency.

## Assigned API, as of v710
- [`AdGui.EnableVideoAds`](https://create.roblox.com/docs/reference/engine/classes/AdGui#EnableVideoAds) (also requires **UI**), [`AdGui.OnAdEvent`](https://create.roblox.com/docs/reference/engine/classes/AdGui#OnAdEvent) (also requires **UI**), [`AdGui.Status`](https://create.roblox.com/docs/reference/engine/classes/AdGui#Status) (also requires **UI**), [`AdPortal`](https://create.roblox.com/docs/reference/engine/classes/AdPortal), [`AdService`](https://create.roblox.com/docs/reference/engine/classes/AdService)
- [`AvatarCreationService.GetBatchTokenDetailsAsync`](https://create.roblox.com/docs/reference/engine/classes/AvatarCreationService#GetBatchTokenDetailsAsync), [`AvatarCreationService.PromptCreateAvatarAsync`](https://create.roblox.com/docs/reference/engine/classes/AvatarCreationService#PromptCreateAvatarAsync) (also requires **AssetCreateUpdate**)
- [`CommerceService`](https://create.roblox.com/docs/reference/engine/classes/CommerceService)
- [`GamePassService`](https://create.roblox.com/docs/reference/engine/classes/GamePassService)
- [`Players.UserSubscriptionStatusChanged`](https://create.roblox.com/docs/reference/engine/classes/Players#UserSubscriptionStatusChanged) (also requires **Players**)
