# LoadOwnedAsset
> [!IMPORTANT]
> This is an undocumented Sandboxable capability. If/when the capability receives [official documentation](https://create.roblox.com/docs/scripting/capabilities), this file will be removed!

## Description
Access to API that loads assets owned by the game creator.

- [`InsertService.LoadAssetVersion`](https://create.roblox.com/docs/reference/engine/classes/InsertService#LoadAssetVersion), - [`InsertService.LoadAsset`](https://create.roblox.com/docs/reference/engine/classes/InsertService#LoadAsset)
