# Consequences
> [!IMPORTANT]
> This is an undocumented Sandboxable capability. If/when the capability receives [official documentation](https://create.roblox.com/docs/scripting/capabilities), this file will be removed!

## Description
Access to API that allows developers to punish players.

## Assigned API, as of v710
- [`BanHistoryPages`](https://create.roblox.com/docs/reference/engine/classes/BanHistoryPages)
- `Players` Members:
  - [`Player.Kick`](https://create.roblox.com/docs/reference/engine/classes/Player#Kick) (also requires **Players**)
  - [`Players.BanAsync`](https://create.roblox.com/docs/reference/engine/classes/Players#BanAsync) (also requires **Players**)
  - [`Players.GetBanHistoryAsync`](https://create.roblox.com/docs/reference/engine/classes/Players#GetBanHistoryAsync) (also requires **Players**)
  - [`Players.UnbanAsync`](https://create.roblox.com/docs/reference/engine/classes/Players#UnbanAsync) (also requires **Players**)
